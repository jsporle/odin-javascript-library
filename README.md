# odin-javascript-library
Full brief available at https://www.theodinproject.com/lessons/node-path-javascript-library

2026.04.15 
For this project I tried to incorporate the following:
- JS Constructor to handle blueprint for books
- JS + CSS Toggle button for sidebar to appear
- JS + CSS Reading status is reflected in individual sidebar on hover 
- HTML + CSS + JS custom alert window (dialogue modal)
- HTML + CSS + JS adding and deleting entries also edits the myLibrary array

For future tweaks I'd consider the following:
- change colour of book card depending on reading status (maybe traffic light system)
- add sort a-z and filter status options
